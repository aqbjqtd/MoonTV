### 🏗️ Dockerfile 优化策略

**5 阶段构建流水线**:

```dockerfile
# 1. base: 基础环境准备
FROM node:20-alpine AS base

# 2. deps: 依赖管理和缓存优化
FROM base AS deps

# 3. builder: 应用构建
FROM base AS builder

# 4. pruned-deps: 生产依赖精简
FROM base AS pruned-deps

# 5. runner: 最终运行时
FROM node:20-alpine AS runner
```

**核心优化技术**:

1. **BuildKit Cache Mounts**: 持久化包管理器缓存
2. **非 root 用户运行**: 安全加固最佳实践
3. **dumb-init 进程管理**: 正确处理信号
4. **健康检查**: 自动监控容器状态
5. **分层缓存**: 最大化构建缓存复用

### 🚀 构建脚本特性

**自动化构建流程**:

- 多架构支持（amd64/arm64）

# MoonTV 项目深度分析报告 (2025-10-02)

## 项目概况
MoonTV 是一个基于 Next.js 14 的现代化影视聚合播放器，支持多源搜索、在线播放、用户管理、多部署方式。项目整体成熟度评分：8.6/10（优秀）

## 核心架构特点

### 1. 多后端存储抽象系统
- **LocalStorage**: 浏览器本地存储（默认）
- **Redis**: 自建Redis服务
- **Upstash Redis**: 云原生Redis服务
- **Cloudflare D1**: 边缘计算SQL数据库
- **统一接口**: 通过 IStorage 接口和 DbManager 类实现透明切换

### 2. 智能配置管理系统
- **双模式配置**: 文件配置(config.json) + 数据库配置
- **动态合并**: 根据部署环境智能合并配置源
- **环境变量**: 作为覆盖和回退机制
- **热更新**: 支持运行时配置变更

### 3. 多层认证架构
- **LocalStorage模式**: 简单密码验证
- **数据库模式**: HMAC签名 + 防重放攻击
- **权限体系**: owner/admin/user三级权限
- **中间件保护**: 路由级别认证控制

### 4. 视频播放引擎
- **主要播放器**: ArtPlayer + HLS.js
- **备用方案**: VidStack React组件
- **高级功能**: Wake Lock、键盘快捷键、智能广告拦截
- **源管理**: 动态源切换和质量检测

# MoonTV 项目身份信息

## 基本信息
- **项目名称**: MoonTV
- **项目类型**: 开源视频聚合平台
- **当前版本**: v3.5.0
- **最后更新**: 2025-12-05
- **项目状态**: 生产就绪，活跃开发中
- **许可证**: MIT License

## 技术栈
- **前端框架**: Next.js 14.2.30 (App Router) + React 18.2.0
- **开发语言**: TypeScript 4.9.5 (严格模式)
- **样式框架**: Tailwind CSS 3.4.17 + Headless UI 2.2.4
- **播放器**: ArtPlayer 5.2.5 + HLS.js 1.6.6 + Vidstack 0.6.15
- **状态管理**: React Context API + SWR/React Query
- **包管理**: pnpm 10.14.0

## 核心特性
- 视频聚合搜索：20+视频源并行搜索
- 多存储后端：LocalStorage/Redis/Upstash/D1
- TVBox集成：动态配置生成和兼容
- 豆瓣数据集成：丰富的电影元数据
- PWA功能：Service Worker离线支持
- 管理面板：用户和视频源管理
- Docker部署：多平台部署支持

## 项目定位
- 目标用户：视频内容聚合需求用户
- 应用场景：个人/家庭视频娱乐中心
- 技术特点：Edge-First架构、全栈Web应用
- 生态系统：兼容TVBox播放器生态
